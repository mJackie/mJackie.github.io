Typecho上的好用插件

Tech
  • typecho

typecho作为国内的博客程序，以其简洁，快速，高效的优点赢得了越来越多用户的青睐，是一个专注于博客的建站程序。
简洁的同时是用户需要更多功能时不得不寻找适合自己的插件，下面推荐一些我一直在用的一些好用的typcho插件：

1.评论过滤器（CommentFilter v1.0.2）
该插件功能齐全，小巧强大，拥有过滤敏感词汇，禁止IP评论等功能，装上该插件，再也不用为垃圾评论，推广链接等发愁了，同事防止恶意留言软件刷留言，占用服务器资源，减少云豆消耗。有了它，连评论验证码都可以直接省掉，方便高效。设置简单，我在这里就不多说了。

2.SAEupload
该插件是针对搭建在sae的typecho上传附件所需插件，启用插件后，只需在设置中输入storage名即可。必备插件。
设置完成后即可在文章编辑页面右侧的附件上传区直接上传附件。

3.友情链接插件（links）
建博客难免需要上几个友链，该插件免去了手写友链的麻烦，可在后台直接添加，管理，删除友链，还可以设置分类，描述，图片。
在主页或需要插入友链的地方插入如下代码即可调用：

<?php if (!empty($this->options->sidebarBlock) && in_array('ShowFriend', $this->options->sidebarBlock)): ?>
<section class="block friend-list">
<p class="ui <?php $this->options->labelColor() ?> ribbon label">友情链接</p>
<div class="animated list friend">
<?php Links_Plugin::output("<a class=\"item\" href=\"{url}\" title=\"{title}\" target=\"_blank\"><div class=\"content\"><p class=\"header\">{name}</p><p>{description}</p></div></a>\n", 10); ?>
</div>
</section>
<?php endif; ?>

4.回复评论邮件及时通知（TimelyNotice）
该插件可实现回复评论后即时邮件通知评论者，一次设置，自动发送，方便博主与用户的互动，增强互动性。

5.Gravatar 头像缓存插件SAE专版（GravatarCache）
gravatar有时会不间断的被强大的中国被K掉，或者访问巨慢。该插件将gravatar头像图片直接缓存到sae的storage，加快头像访问速度，防止打开缓慢。

6.自动为文章中出现的关键词添加链接（Keywords）
懂点seo的人都懂，锚文本的多少直接影响网站权重，站内锚文本更利于搜索引擎的收录。文中若出现与本站标签相同的关键词则自动添加标签页链接。同时还可设置出现频率，还可以自定义关键词。

7.Google高亮代码（Google Code Prettify）
typecho自带的代码显示是白底红字，没有关键字高亮的，该插件提供5中代码高亮风格，支持多种语言关键字。还可自定义css样式风格

8.无缝集成HighSlide双核版实现自动化弹窗与页面相册功能（HighSlide）
该插件有基础版和全功能版，功能强大，实现相册弹窗，支持插图幻灯/html弹窗/页面相册等。

9.嵌入HTML5+Flash视频播放器JW Player6（JWPlayer）
JWplayer是老牌开源页面播放器，与HighSlide一样有公司化的商业模式，功能也份大量足毫不含糊。该插件可指定播放器宽度，封面，风格，字幕，标题，多视频列表播放等功能。界面简洁小巧精致。

10.数据备份SAE版（DbManager）
该插件可一键备份数据库，方便快捷，拥有备份，导入，优化等功能。可备份至本地或服务器。还可选择单独表，进行单独的备份。
