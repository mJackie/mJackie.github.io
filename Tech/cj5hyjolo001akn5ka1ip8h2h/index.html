快速切换IP地址

Tech
  • cmd

辗转于宿舍与实验室，不同的网线需要不同的IP地址。将以下代码用记事本保存为bat格式文件即可实现快速切换IP。
注意：以管理员身份运行，否则会报错。

自动获取IP：

@echo off
cls
Echo **********************************************************
Echo           正在修改IP地址和DNS服务器地址,请耐心等待…………
Echo **********************************************************
netsh interface ip set address name="以太网" source=dhcp
netsh interface ip delete dns "以太网" all
::将“以太网”换为网络连接名称
ipconfig /flushdns
ipconfig /all
Echo **********************************************************
Echo          OK！！已修改成功！请按任意键退出…………
Echo          谢谢您的使用!Design By Liu
Echo **********************************************************
Pause

设置固定IP：

@echo off
cls
Echo **********************************************************
Echo           正在修改IP地址和DNS服务器地址,请耐心等待…………
Echo **********************************************************
cmd /c netsh interface ip set address name="以太网" source=static addr=125.219.42.14 mask=255.0.0.0 gateway=125.219.42.1 gwmetric=1
::设置你的IP地址，子网掩码，默认网关
cmd /c netsh interface ip set dns name="以太网" source=static addr=202.196.96.131
::设置dns
cmd /c netsh interface ip add dns name="以太网" addr=8.8.8.8 index=2
::设置备用dns
ipconfig /all
Echo **********************************************************
Echo          OK！！已修改成功！请按任意键退出…………
Echo          谢谢您的使用！Design By Liu
Echo **********************************************************
Pause
