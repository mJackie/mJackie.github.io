姿态角

Tech

载体的姿态是载体坐标系在地理坐标系的表示，常用航向角、俯仰角和横滚角作为载体的姿态角。基于地理坐标系及载体坐标系，可得出姿态角的定义：


航向角：载体的x b 轴在水平面的投影与地磁北极夹角，叫做航向角，其表示范围为 0～360°；


俯仰角：载体的x b 轴在当地水平面的投影与x b 间的夹角，叫做俯仰角，范围为-90～90°；


横滚角：载体的y b 轴与其在当地水平面的投影间的夹角，叫做横滚角，范围为-180～180°。

北京蓝桥之行

Jotting

想起当初报名还是在上个学期过年以前，当时还在考虑要不要交300块钱报个名，自己对于算法，数据结构也是一窍不通，好在在老师的鼓励下，和寝室的张一块报了名。倘若当初没有报名，也就不会有了这次北京之行，更不会得奖，所以万事都要积勇于尝试，争取，正如《高效能人士的七个习惯》中的第一个习惯——积极主动。

寒假在实验室呆了几天，太闷了，实在待不下去了，和张晚上看了《霍比特人3》，第二天背着书包便回家了。当当上买了一本java的数据结构书，想着寒假好好看看，可惜计划赶不上变化，过年事也多，整个假期过去了，也没看几页(其实看了也没多大用，蓝桥杯主要考算法，还是多练题有效果)。

4月份在郑州轻工业学院省赛，根本没有准备，也就前一个晚上看了几道历年题，想着能拿个奖就不错了。一星期后结果出来了，省一等奖。

5月29日晚上9点35分，迈上了一直向往的城市——北京的第一步。

去的一路上人不多，三国杀，大富翁，又看了一部电影，很快就过了。

去之前的前一个星期，课余时间都在做官网上的练习题，一共做了大概30道题。

第二天早上吃完饭9点多，时间不太够，也没有出去浪，比赛是在下午2点到6点，考试还发了一杆笔，一盒奶，一瓶水，一个面包，很贴心。要说的就是一定不能浪费太多时间在一道题上，有些弄不懂的越扣越浪费时间，还不如直接跳过。

比赛完坐地铁直接去了天安门，正巧赶上降旗仪式，人很多，也看不太清楚，郁闷的是天安门广场有围栏护着，不让进。我们几个绕着马路走了一路，又到王府井小吃街逛了一圈，买了一串糖葫芦，10块。

晚上回去在郭房间三国杀，也没看时间直接到了一点多。王去找他舅了，我拿着房间钥匙，回房间路上想着阮不会在门外等吧，还好他找楼下的开了门，进了房间。

第二天，阮提前去清华听讲座了，等王回来后，一块做地铁去奥体中心绕着鸟巢，水立方压马路。

下午颁奖典礼在北大邱德拔体育馆举行，河大民生学院的一个学生和曾经教过我的吴老师竟然得了特等奖，很赞。水木年华的缪杰唱了《为梦而生》，《一生有你》。

二等奖，也算是自己之前想象的最好的结果了，很满意。郝和郭一等奖，很赞。

晚上火车正好赶上了农民工返乡，人超多，难受。

也许是自己之前期望太高，眼中的北京并没有想象中的好，其实天南海北都是一样的，就像人与人之间，马云和我也一样长着两个眼睛一张嘴嘛。

天外有天，楼外楼，自己要多加努力，勤奋上进。
最后还是要抓住眼前，从现在做起，英语六级，网络作业，期末考试……

Git学习笔记

Tech
  • Git

Git是一款免费、开源的分布式版本控制系统，用于敏捷高效地处理任何或小或大的项目。分布式的git相比于集中式的svn最大区别在于开发者可以提交到本地，每个开发者通过克隆（git clone），在本地机器上拷贝一个完整的Git仓库。
git构架上分为3个区，工作区，暂存区与版本库。用户在工作区的更改通过git add命令添加至暂存区，暂存区的数据通过git commit提交到版本库。用户通过git checkout从版本库中检出数据。
git的远程协作需要先fork开源项目，然后本地git clone该项目，通过git pull，git push编辑项目，通过git remote获取原作者的更新，pull request为该项目做贡献。

git 双击table自动完成

The First ASM

Tech
  • 汇编

写的第一个汇编程序，发出来纪念一下。

;试编写程序，完成下面公式的计算。
;A ←（X-Y+24）/ Z 的商，B ←（X-Y+24）/ Z 的余数
;其中，变量 X 和 Y 是 32 位有符号数，变量 A、B、Z 是 16 位有符号数。

DATA SEGMENT
X DD 481
Y DD 240
A DW ?
B DW ?
Z DW 24
DATA ENDS

CODE SEGMENT
    ASSUME DS: DATA, CS: CODE
START:
    MOV AX, DATA
    MOV DS, AX
    MOV AX, word ptr ds:[0]
    MOV DX, word ptr ds:[2]
    MOV BX, word ptr ds:[4]
    MOV CX, word ptr ds:[6]
    SUB AX, BX
    SBB DX, CX
    ADD AX, 24
    ADC DX, 0
    MOV BX, Z
    IDIV BX
    MOV A,AX
    MOV B,DX

    MOV AX, 4C00H
    INT 21H
CODE ENDS
END START
